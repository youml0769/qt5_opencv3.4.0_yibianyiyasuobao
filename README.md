## 使用步骤

1. **下载资源**：从仓库页面下载提供的压缩包。
2. **解压缩**：将压缩包解压到您希望存放开发环境的目录下。
3. **环境配置**：
   - 在QT Creator中新建或打开项目。
      - 配置Qt项目的.pro文件，添加OpenCV和QT的相关路径。
         - 确保系统路径包含OpenCV的bin目录，以便程序运行时能找到所需的DLL文件。

            4. **开始开发**：现在您可以直接在项目中使用OpenCV的功能了，无需单独编译OpenCV库。
